# 일정관리 앱

### 일정관리하는 앱입니다.
- 일정 등록, 조회, 전체조회, 작성자로 조회, 일정 수정(제목, 작성자), 일정 삭제
- 일정 등록시 비밀번호를 설정합니다.
- 수정 및 삭제시 비밀번호가 필요합니다.
- 일정 조회시 수정일 순으로 나열됩니다.
